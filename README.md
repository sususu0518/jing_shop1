# 晶酱蔬菜商城

> 蔬菜商城是我自己利用业余时间编写的一个项目，用户在商场选购蔬菜下单。可以在网页留言、浏览文章。
后台管理员可以查看订单、上架蔬菜、查看用户留言、发布文章等。另外还接入了支付宝沙箱支付，模拟付款。

> 后台管理系统可以一键生成代码和页面，帮你更快完成任务。

#### 采用的技术
1. Springboot、Redis
2. Freemarker、Layui、Echarts
3. Mybatis-plus
4. Mysql8

#### 模块
| 模块  | 备注           |
|-----|--------------|
| 首页  | 二级分类、热销蔬菜、文章 |
| 我的  | 订单、住址        |
| 文章  | 文章阅读         |
| 留言板 | 客户留言         |
| 商城  | 搜索页          |
| 后台管理  | 报表、留言、蔬菜管理          |

#### 另外商城数据采用爬虫抓取清洗入库
